# Code of Conduct for {{PROJECT_NAME}}

We value a welcoming and inclusive community at {{PROJECT_NAME}}. All contributors, maintainers, and users are expected to follow this Code of Conduct.

## 1. Our Standards

We expect everyone to:

- Be respectful and considerate.
- Use welcoming and inclusive language.
- Be patient and collaborative.
- Avoid harassment, discrimination, or offensive behavior.
- Respect privacy and personal boundaries.

## 2. Unacceptable Behavior

Unacceptable behavior includes, but is not limited to:

- Harassment, intimidation, or bullying.
- Discriminatory comments related to race, gender, sexual orientation, religion, or any other protected class.
- Threats of violence or personal attacks.
- Publishing private information without consent.

## 3. Reporting Issues

If you experience or witness any unacceptable behavior, please report it by contacting: **[hr@monark.io](mailto:hr@monark.io)**.
All reports will be handled **confidentially**.

## 4. Enforcement

Maintainers are responsible for enforcing this Code of Conduct. Consequences for violating the Code may include:

- Warnings and/or temporary removal from the community.
- Removal of contributions or blocking access.
- Reporting to relevant authorities in extreme cases.

## 5. Scope

This Code of Conduct applies to all spaces related to {{PROJECT_NAME}}, including:

- Public and private repositories.
- Issue trackers and discussion forums.
- Meetings, events, and social media channels associated with the project.

## 6. Acknowledgements

This Code of Conduct is adapted from the [Contributor Covenant](https://www.contributor-covenant.org/), version 2.1, with modifications for **Monark Inc.**.

Thank you for helping us maintain a **positive and inclusive community**!
